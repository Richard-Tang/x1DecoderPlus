## 介绍

AntSword(蚁剑)全参数流量XOR+Base64，基于[AntSword-Cryption-WebShell](https://github.com/xiaopan233/AntSword-Cryption-WebShell/tree/main/php)基础上进行改造，主要用来逃避一些流量检测。

## 使用
